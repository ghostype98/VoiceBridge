## 生产环境代码更新流程（示例）

> 将 `YOUR_REPO_URL`、`/opt/voicebridge` 等替换为实际值。

### 前置信息

- **项目路径**：`/opt/voicebridge`
- **Git 远程**：`origin  https://github.com/YOUR_ORG/voicebridge.git`
- **服务管理**：systemd，服务名 `voicebridge`

### 更新步骤

```bash
cd /opt/voicebridge
git fetch origin
git checkout main
git reset --hard origin/main
sudo systemctl restart voicebridge
sudo systemctl status voicebridge --no-pager
curl -s http://127.0.0.1:8002/health
```

### 配置注意

- 本地 `config/config.yaml` 不应被 git 覆盖；更新前备份该文件。
- 环境变量 `DB_*`、`ALIYUN_*` 建议在 systemd unit 或 `.env` 中配置。
