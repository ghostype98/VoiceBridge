📱 移动端调试工具

实时预览和调试移动端页面效果

⚠️ 重要提示：这是开发调试工具，不是用户访问的页面！
用户最终访问的页面是：/login 或 /interview
📱 手机视图 (375 x 667) 🔄 刷新

💡 使用说明

  • 设备切换：点击上方按钮切换不同设备视图
  • URL修改：修改URL输入框可以预览不同页面
  • 实时刷新：点击刷新按钮重新加载页面
  • 浏览器调试：按 F12 打开开发者工具进行深度调试
  • 移动端调试：使用"在手机浏览器打开"获取二维码，用手机扫描访问
  • 响应式测试：调整浏览器窗口大小测试响应式效果

📌 页面访问说明

  • 调试工具页面：/mobile-debug - 仅用于开发调试，用户不要访问此页面
  • 登录页面：/login - 用户最终访问的登录页面
  • 面试页面：/interview - 用户登录后访问的面试页面
  • 为什么不一样：调试工具页面是一个包装页面，里面用iframe嵌入实际页面。直接访问实际页面会看到完整的页面效果。

🔧 浏览器开发者工具移动端模拟器

  • Chrome/Edge：按 F12 → 点击设备图标（📱）或按 Ctrl+Shift+M
  • Firefox：按 F12 → 点击响应式设计模式图标或按 Ctrl+Shift+M
  • Safari：启用开发者菜单 → 开发 → 进入响应式设计模式
  • 设备预设：可以选择 iPhone、iPad、Android 等预设设备
  • 自定义尺寸：可以手动输入宽度和高度
  • 网络模拟：可以模拟慢速网络、离线状态等
